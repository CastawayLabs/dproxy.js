dproxy
======

Fast, Lightweight, Dynamic.

node.js reverse proxy driven by `redis`.

How is it faster
----------------

- `dproxy` abstracts from http to raw tcp

How it works
------------

Features:
---------

- Management Web Interface
- Production-tested at NodeGear
- [x] HTTPs-ready
- [x] Websocket support
- [x] SSL Termination
- [x] Load Balancing
- [x] Dynamic Routing
